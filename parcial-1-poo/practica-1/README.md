# Práctica 1: Clases y Encapsulamiento

## Objetivo
Crear una clase Usuario con atributos privados y métodos getters/setters.

## Clase Usuario
- **Atributos privados:** nombre, correo
- **Métodos:** __construct(), getNombre(), getCorreo(), setNombre(), setCorreo()

## Instrucciones de ejecución
1. Colocar Usuario.php e index.php en la misma carpeta
2. Ejecutar con XAMPP
3. Ver la salida en el navegador

## Ejemplo de salida
Nombre: Gerson
Correo: gerson@gmail.com
Nombre actualizado: José
Correo actualizado: jose@gmail.com
